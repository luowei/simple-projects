#admin-example

一个spring mvc的轻量级web项目...

**技术包含:**

1. spring mvc,spring jdbc ;
2. 页面数据加载全部使用ajax请求;
3. 直接使用ModelAndView导出Excel数据视图;
4. 对导出的Excel文件作修改后，再导入执行可批量更新记录，同时更新多张表的记录;
